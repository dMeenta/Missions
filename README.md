To run this proyect locally:

----------------------------------Database----------------------------------
Run *db_commands.sql* on your postgreSQL database manager
create an *.env* on *./server* where you have to specify DB_USER, DB_HOST, DB_PASSWORD, DB_PORT, DB_DATABASE like this: *DB_USER=postgres*
----------------------------------Server----------------------------------
1.- cd server
2.- npm i
3.- npm run dev
----------------------------------Client----------------------------------
1.- cd client
2.- npm i
3.- npm run dev
